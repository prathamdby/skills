# Structures to Avoid

## Binary Contrasts

These create false drama by framing a negation first.

| Pattern                          | Fix                     |
| -------------------------------- | ----------------------- |
| not X, but Y                     | state Y directly        |
| isn't X, it's Y                  | state Y directly        |
| not because X, but because Y     | state the actual reason |
| the question isn't X, it's Y     | ask Y directly          |
| stops being X and starts being Y | describe the change     |

**Instead:** "The problem is Y." Drop the negation entirely.

## Negative Listing

Listing what something is _not_ before revealing what it _is_.

| Pattern                             | Fix     |
| ----------------------------------- | ------- |
| Not a X. Not a Y. A Z.              | state Z |
| It wasn't X. It wasn't Y. It was Z. | state Z |

**Instead:** State Z. The reader doesn't need the runway.

## Dramatic Fragmentation

Sentence fragments for emphasis read as manufactured profundity.

| Pattern                            | Fix                       |
| ---------------------------------- | ------------------------- |
| Noun. That's it. That's the thing. | complete sentence         |
| X. And Y. And Z.                   | combine or vary structure |
| This unlocks something. Word.      | complete the thought      |

**Instead:** Complete sentences. Trust content over presentation.

## Rhetorical Setups

These announce insight rather than deliver it.

| Pattern             | Fix               |
| ------------------- | ----------------- |
| What if [reframe]?  | make the point    |
| Here's what I mean: | state it directly |
| Think about it:     | let readers think |
| And that's okay.    | delete            |

## False Agency

Giving inanimate things human verbs.

| Pattern                | Fix                                 |
| ---------------------- | ----------------------------------- |
| the data suggests      | someone read the data and concluded |
| the decision emerges   | someone decided                     |
| the culture shifts     | people changed behavior             |
| the market rewards     | buyers paid                         |
| the conversation moves | someone steered it                  |

**Instead:** Name the human. "The team fixed it" beats "the complaint becomes a fix."

## Passive Constructions

Every sentence needs a subject doing something.

| Pattern                  | Fix                  |
| ------------------------ | -------------------- |
| X was created            | name who created it  |
| it is believed that      | name who believes it |
| mistakes were made       | name who made them   |
| the decision was reached | name who decided     |

## Uniform Rhythm

| Pattern                            | Fix                  |
| ---------------------------------- | -------------------- |
| three-item lists                   | use two items or one |
| consecutive sentences equal length | break one            |
| every paragraph ends punchily      | vary endings         |
| staccato short sentences           | let one breathe      |

## Em Dashes

AI text overuses em dashes to join clauses without committing to sentence boundaries.

| Pattern                                      | Fix                                                         |
| -------------------------------------------- | ----------------------------------------------------------- |
| clause — standalone follow-up                | period: `clause. Follow-up.`                                |
| clause — same-sentence aside or continuation | comma: `clause, aside or continuation.`                     |
| multiple em dashes in one paragraph          | work through each; mix periods and commas as grammar allows |

**Instead:** No em dashes in the final draft. If the second half can stand alone, start a new sentence. If it cannot, keep one sentence with a comma.
